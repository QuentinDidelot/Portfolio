EcoGarden

Description du projet :

EcoGardenApi est une API développée pour une entreprise spécialisée dans le jardinage écologique. Cette API permet de rendre disponibles des informations sur le jardinage et la météo, dans le but de les partager avec des partenaires et associations, facilitant ainsi l’accès à des conseils pratiques pour les amateurs de jardinage.

Fonctionnalités :

◎ Créer et gérer des comptes utilisateurs avec des informations essentielles (comme la ville).

◎ Authentification sécurisée via un système de tokens JWT pour l'accès aux fonctionnalités.

◎ Accéder aux conseils de jardinage mensuels pour des recommandations adaptées.

◎ Obtenir des informations météorologiques pour une ville donnée afin de mieux planifier les activités de jardinage.

◎ Intégrer la météo locale pour la ville de l'utilisateur afin d'affiner les conseils de jardinage.

◎ Ajouter, modifier et supprimer des conseils de jardinage en fonction des besoins des administrateurs.

◎ Gérer les utilisateurs, y compris la mise à jour de leurs informations ou leur suppression.

◎ Gestion robuste des erreurs et réponses appropriées pour assurer une utilisation fluide et sécurisée.

Technologies utilisées :

◎PHP 8.3

◎Symfony 6.4

◎MySQL

◎API Rest (Création)

◎API publique (Utilisation)

◎Token JWT

◎Nelmio
